Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je programovací jazyk, který se používá pro vytváření interaktivních webových stránek.
  • Může být využit pro dynamickou manipulaci s obsahem stránky, jako je například přidání nebo odstranění HTML prvků na základě uživatelských akcí.
  • JavaScript je také základem pro vývoj moderních webových aplikací, které využívají technologie jako je React, Vue.js nebo Angular.
  • JavaScript může být použit pro tvorbu vizualizací dat, například pomocí knihovny D3.js.
  • Další využití JavaScriptu zahrnuje vývoj mobilních aplikací pomocí frameworků jako je React Native nebo vývoj serverových aplikací pomocí Node.js.

Zajímavé příklady využití JavaScriptu najdete zde:

  • Three.js - knihovna pro tvorbu 3D grafiky
  • p5.js - knihovna pro tvorbu kreativních grafických aplikací
  • Node.js - platforma pro běh JavaScriptu na serveru

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Use strict equality (===) to avoid unexpected type coercion.
  2. Declare variables with `let` and `const` to ensure block scope.
  3. Always end statements with a semicolon to avoid automatic semicolon insertion (ASI) issues.
  4. Keep functions small and focused on a single task.
  5. Avoid global variables by encapsulating code within functions or modules.
  6. Comment your code to explain complex logic or important decisions.

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

<!-- Vloží script do konzole -->
<script>console.log('Hello');</script>

<!-- Vloží externí soubor main.js do HTML dokumentu -->
<script src="main.js"></script>

<!-- Spustí kód po načtení stránky -->
<body onload="console.log('Stránka načtena')"></body>

<!-- Při kliknutí na tlačítko spustí kód -->
<button onclick="alert('Tlačítko bylo kliknuto')">Klikni mě</button>

<!-- Vytvoří nový script element a vloží ho do DOM -->
<script>
let script = document.createElement('script');
script.innerHTML = "console.log('Dynamicky vložený script')";
document.head.appendChild(script);
</script>

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %=, **= (přiřazení a operace současně)

Logické

&& (logické AND), || (logické OR), ! (logické NOT)

Porovnávací

== (rovná se), === (přísná rovnost - rovná se a stejný typ), != (nerovná se), !== (přísná nerovnost - nerovná se nebo jiný typ), > (větší než), < (menší než), >= (větší nebo rovno), <= (menší nebo rovno)

Bitové

& (bitové AND), | (bitové OR), ^ (bitové XOR), ~ (bitové NOT), << (bitový posun vlevo), >> (bitový posun vpravo), >>> (bitový posun vpravo s nulovým doplněním)

Typové

typeof (typ proměnné), instanceof (kontrola instance objektu)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Zobrazí jednoduché varovné okno s daným textem a tlačítkem OK */ alert('Nezadal jsi správné heslo!');

prompt()

/* Zobrazí dialogové okno s textovým polem pro zadání vstupu uživatelem */ let password = prompt('Zadejte své heslo:');

confirm()

/* Zobrazí dialogové okno s textem, tlačítky OK a Cancel. Vrací true nebo false podle zvolené možnosti. Pokud uživatel klikne na OK, přesměruje ho na zadanou URL. */ if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování: Popup okna mohou být rušivá a nepříjemná pro uživatele, což může vést k negativnímu uživatelskému zážitku. Přílišné používání popup oken může způsobit, že uživatelé začnou ignorovat důležitá oznámení nebo varování. Popup okna mohou být blokována některými prohlížeči nebo pluginy, což může způsobit, že důležité informace nebudou zobrazeny.
